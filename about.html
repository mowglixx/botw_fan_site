About this site

This website was developed as a proof of knowledge after 2 weeks of CodeNation training.

The site is a demo of what can be achieved without JavaScript and all styles including aesthetic icons have been created using custom fonts and CSS3 Features, one such feature is the :after pseudo element

Triforce Icons in nav

.triforce {
  font-family: "Triforce";
  color: var(--triforce-gold);
  }
.triforce:after {
  content: "*";
  }

This approach made the site more accessible to screen readers by not having unnecessary characters on the screen similar to the way FontAwesome provides their fonts, each .icon has a defining style with the icon color and font and the :after defines the symbol being used to conjure up the icon from the selected font without adding characters to the HTML.
